HTML

  • Remember to link your JS and CSS files in your HTML file;
  • Alt + Shift + F for indentation and better comprehension of the code;
  • Insert div copied inside another parent div;
  • Add you images in the "src" attribute inside the img tags;

CSS

  • Consider having a CSS file specific for the Carousel;
  • Remember to link your CSS file in your HTML file;
  • Parent div should have overflow: hidden and same aspect-ratio that was chosen;
  • You can change the CSS values with the variables in "root" pseudo-class.

JS

  • Consider having a JS file specific for the Carousel;
  • Remember to link your JS file in your HTML file;
